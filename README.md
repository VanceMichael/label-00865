# 赛博战机：终极防御

## 1. 运行方式

### 本地开发
直接在现代浏览器中打开 `frontend-admin/index.html`

### Docker 部署
```bash
# 使用 docker-compose 构建并运行
docker compose up -d --build

# 访问 http://localhost:8080
```

## 2. 服务说明

- **前端服务**: Nginx 托管的静态 HTML5 Canvas/JS 应用（端口 8080）

## 3. 测试账号

这是一个独立的客户端游戏，无需登录。
- **数据持久化**: 最高分数存储在浏览器本地存储（`localStorage`）中。

### 操作说明
- **PC端**: 方向键 ← → 控制左右移动
- **移动端**: 触屏左右滑动控制（自动检测设备类型并显示对应提示）
- **射击**: 自动发射，无需手动触发

### 敌机类型
| 类型 | 外观 | 生命值 | 得分 | 特殊能力 |
|------|------|--------|------|----------|
| 普通敌机 | 霓虹紫描边 | 1 | 100分 | 无 |
| 快速敌机 | 荧光绿描边 | 1 | 200分 | 高速移动 |
| 重装敌机 | 科技红描边 | 3 | 500分 | 发射敌弹 |
| 精英BOSS | 三色霓虹 | 20 | 3000分 | 8方向散射弹幕 |
| 终极BOSS | 电流光效 | 50 | 10000分 | 双模式攻击 |

### 终极BOSS双模式攻击
- **模式1 - 跟踪子弹**: 发射紫色跟踪子弹追踪玩家，同时散射普通弹幕
- **模式2 - 旋转激光**: 4条红色激光从BOSS中心向外旋转扫射
- 每5秒自动切换攻击模式，BOSS头顶显示当前模式

### 道具系统
| 图标 | 名称 | 效果 |
|------|------|------|
| W | 火力升级 | 三发散射，持续8秒 |
| Sh | 护盾防护 | 免疫1次伤害 |
| Li | 生命补给 | +1生命值 |
| L | 激光武器 | 穿透激光，持续5秒 |

### 得分与成长系统
- **连杀奖励**: 10秒内连续击毁5架敌机触发"连杀加成"，额外+500分
- **连杀提示**: 3杀、4杀时显示进度提示，5杀时显示"🔥 连杀加成!"
- **等级提升**: 每5000分升1级，显示全屏霓虹"LEVEL UP"特效
- **射速提升**: 等级提升时射速小幅提升（最高2倍）

### 音效系统
- **背景音乐**: 赛博朋克风格电子音乐，使用Web Audio API合成
- **节奏递进**: BGM节奏随等级提升加快（120BPM → 最高180BPM）
- **音量控制**: 默认50%，支持滑块调节，关闭音效时隐藏滑块
- **事件音效**: 射击、爆炸、道具拾取、受伤、BOSS登场等

### 难度递进
| 分数区间 | 敌机类型 | 刷新间隔 | 速度倍率 |
|----------|----------|----------|----------|
| 0-5000 | 普通+快速 | 2秒 | 1x |
| 5000-15000 | +重装敌机 | 1.2秒 | 1.5x |
| 15000+ | 全类型高频 | 0.8秒 | 2x |

### 特效系统
- **BOSS登场**: 屏幕震动 + 霓虹边框闪烁
- **等级提升**: 全屏霓虹"LEVEL UP"特效，蓝紫色发光动画
- **子弹特效**: 尾迹光效，激光为持续霓虹光束
- **爆炸特效**: 霓虹粒子炸开动画
